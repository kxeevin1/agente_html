Introducción

El Algoritmo de Kruskal es un algoritmo que se utiliza para encontrar el árbol de expansión mínima (TEM) en un grafo no dirigido y conexo. Este algoritmo es fundamental en la teoría de grafos y tiene aplicaciones prácticas en redes, mapas, y optimización.

Metodología

El algoritmo de Kruskal se basa en el siguiente procedimiento:

  1. Ordenar los arcos del grafo por su peso (o costo) ascendente.
  2. Inicializar un conjunto disjunto para cada vértice del grafo, donde cada vértice es su propio conjunto.
  3. Para cada arco en el orden de pesos:
    1. Si los extremos del arco pertenecen a conjuntos diferentes, unirlos en un solo conjunto.
    2. Agregar el arco al árbol de expansión mínima.
  4. Repetir hasta que se hayan agregado todos los vértices del grafo al árbol de expansión mínima.

Implementación

A continuación, se muestra una implementación básica del Algoritmo de Kruskal en JavaScript:


            function kruskal(graph) {
                let edges = [];
                for (let i = 0; i < graph.length; i++) {
                    for (let j = i + 1; j < graph.length; j++) {
                        if (graph[i][j] !== 0) {
                            edges.push({ source: i, target: j, weight: graph[i][j] });
                        }
                    }
                }

                edges.sort((a, b) => a.weight - b.weight);

                let parent = Array(graph.length).fill(null);
                let rank = Array(graph.length).fill(0);

                function find(x) {
                    if (parent[x] === null) return x;
                    return parent[x] = find(parent[x]);
                }

                function union(x, y) {
                    let rootX = find(x);
                    let rootY = find(y);
                    if (rootX !== rootY) {
                        if (rank[rootX] > rank[rootY]) {
                            parent[rootY] = rootX;
                        } else if (rank[rootX] < rank[rootY]) {
                            parent[rootX] = rootY;
                        } else {
                            parent[rootY] = rootX;
                            rank[rootX]++;
                        }
                    }
                }

                let mst = [];
                for (let edge of edges) {
                    if (find(edge.source) !== find(edge.target)) {
                        mst.push(edge);
                        union(edge.source, edge.target);
                    }
                }

                return mst;
            }

            const graph = [
                [0, 2, 4],
                [2, 0, 1],
                [4, 1, 0]
            ];

            console.log(kruskal(graph));

Resultados

El algoritmo de Kruskal encuentra el árbol de expansión mínima para un grafo dado. En el ejemplo anterior, el resultado es:


            [
                { source: 0, target: 1, weight: 2 },
                { source: 1, target: 2, weight: 1 }
            ]

Conclusiones

El Algoritmo de Kruskal es un algoritmo eficiente para encontrar el árbol de expansión mínima en grafos no dirigidos y conexos. Su implementación en JavaScript muestra que es fácil de entender y aplicar, lo que lo convierte en una herramienta valiosa en la teoría de grafos.
