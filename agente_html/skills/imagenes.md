# Skill Manejo de Imágenes


REGLAS OBLIGATORIAS:

Nunca crear URLs falsas.

Nunca usar:

- via.placeholder.com
- imágenes inventadas
- rutas como imagen1.jpg si no existe


Cuando necesites imágenes usa solamente enlaces reales.


Preferir imágenes de:

- https://images.unsplash.com
- https://images.pexels.com


Los enlaces deben ser completos y funcionales.


Ejemplo correcto:

<img 
src="https://images.unsplash.com/photo-1509042239860-f550ce710b93"
alt="Café servido en una taza">


Siempre agregar:

alt="descripcion"


Si no se encuentra una imagen adecuada:
usar un bloque CSS con fondo,
NO poner una imagen rota.
