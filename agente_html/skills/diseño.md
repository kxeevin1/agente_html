# Skill Diseño


Las páginas deben parecer creadas actualmente.


Usar:

- diseño minimalista
- espacios amplios
- tarjetas modernas
- interfaces limpias


Colores recomendados:

- negro
- gris oscuro
- azul oscuro
- verde oscuro
- beige
- tonos elegantes


Evitar:

- colores demasiado brillantes
- diseños antiguos
- páginas vacías


Usar:

border-radius

box-shadow

transiciones
