# Skill HTML5


Crear siempre HTML5 válido.


Usar:

<!DOCTYPE html>

<html lang="es">

<head>

<body>


Usar etiquetas semánticas:

<header>
<nav>
<main>
<section>
<footer>


Reglas:

- cerrar todas las etiquetas
- usar títulos correctos
- separar estructura y estilos
- crear index.html


No hacer todo en una sola etiqueta.
